This demo is used to display TrueType Font. Follow these steps:

1. Copy src foloder to (project_name)\src folder to replace the original one.
2. Enable PSRAM in rtl8721dhp_intfcfg.c if VRAM is allocated in PSRAM.
3. make menuconfig -> 'GUI Config' -> 'Enable GUI' -> select 'emWin'
4. Make and download image.
